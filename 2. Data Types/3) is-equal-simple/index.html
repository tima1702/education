Data Types/is-equal-simple

Задача:

Написать функцию, которая будет сравнить объекты

Пример

isEqual({}, {}) === true
isEqual({ a: 'test' }, { a: 'test' }) === true
isEqual({ a: 'test' }, { b: 'test' }) === false
isEqual({ a: 'test', b: null }, { a: 'test', b: 'null' }) === true
isEqual({ a: 'test' }, { a: 'test1' }) === false
isEqual({ a: null }, { a: null }) === true
isEqual({ a: undefined }, { a: null }) === false
isEqual({ a: undefined }, { a: undefined }) === true

Замечание:

Значение объекта не может быть объектом
