Задача:

Задача Написать TokenService, в котором есть слудеющие функции subscribe(callback) - добавляет подписку на изменение токена (параметром будет новый токен, или null, если токен удален) setToken(token) - устанавливает значения токена (вызывает подписчиков) removeToken() - удаляет токен (вызвает подписчиков на изменеия) getToken() - возващяет токен

Примеры

const testService = new TokenService();
testService.subscribe((data) => { const message = data ? 'subscriber: ' + data : 'token was removed'; console.log(message); });
testService.setToken('newToken'); // log: "subscriber: newToken"
testService.getToken(); // "newToken"
testService.removeToken(); // log: "token was removed" null
