Basic/string-to-type

Задача:

Написать функцию которая будет преобразовывать строку в нужному типу

Пример

stringToType('null') => null
stringToType('undefined') => undefined
stringToType('true') => true
stringToType('100') => 100
stringToType('') => ''
stringToType('100ab0') => '100ab0'
stringToType('abcd') => 'abcd'
stringToType('0') => 0

Замечание:

Обратите внимание, что в агрументах строка, а возвращяемый результат не всегда должен быть строкой
